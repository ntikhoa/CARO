How to control:

Press 'W A S D' key to move.
Press 'Enter' to mark icon.
Press 'L' to save game.
Press 'U' to undo move.
Press 'ESC' key to exit games.

Mode 0: Traditional.
-Two player will mark there icon on the board in turn.
-Who have 5 icons consecutively first will win
-The winning can be in row, colume and cross

Mode 1: A holes
-Game rules like mode 0.
-The board will have some gaps, and both player cannot mark there.

Mode 2: Think for both
-Game rules like mode 0.
-After each 4 turns, Two player icons on the board will be swap.
-The represent icon will be the same.

Mode 3: The bomb.
-Game rules like mode 0.
-There will be some bomb on the board 'B'
-Who has more than 4 icons surround the bomb will lose. 

Mode 4: TIC TAC TOE
-Game rules like mode 0 but with 3*3 board.
-It is a Western Caro version.

Mode 5: The neutral point
-Game rules like mode 0.
-There will be some advance point on the board 'A'
-The 'A' point will count for both player

Mode 6: A second chance
-Game rules like mode 0.
-But both player need to have 5 consecutive icons two times
-Or 6 consecutive icons one times

Mode 7: One cry, One laugh
-Game rules like mode 0.
-After each 5 turns the game will choose a place on the board to delete icons
-There would be nothing happening if the place have no icon.

Mode 8: Modern rules
-Game rules like mode 0.
-But if the winning is blocked at two side by the component, the winning will not be counted.
